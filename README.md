# Chatbot Jurídico 🏛️

Asistente de consultas jurídicas especializado en jurisprudencia colombiana sobre violencia intrafamiliar y debida diligencia estatal.

## Stack
- Python + Flask
- Gemini 2.5 Flash Lite (Google AI)
- HTML/CSS/JS vanilla

## Requisitos
- Python 3.12
- API Key de Google AI Studio

## Instalación local

```bash
git clone https://github.com/tu-usuario/tu-repo
cd chatbot
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Crear variable de entorno:
```bash
set GOOGLE_API_KEY=tu_api_key_aqui
```

Correr:
```bash
py app.py
```

Abrir `http://localhost:5000`
